# 05 – Arborescence : ranger les fichiers

Image simple
- Un site = un dossier avec des fichiers (index.html, images, styles).
- Si on déplace ou renomme un fichier sans prévenir, l’image casse ou la page ne se charge pas.

Mini-exercice (5 minutes)
1. Créer un dossier « mon_premier_site » sur le bureau.
2. Créer un fichier texte, le renommer en `index.html`.
3. Ouvrir `index.html` dans le navigateur et écrire simplement « Bonjour ».
4. Ajouter une image dans le même dossier et l’afficher avec `<img src="mon_image.jpg">`.

Messages clés
- Le chemin du fichier doit être exact (nom + extension).
- Ranger clairement évite 80 % des erreurs débutantes.

Idée d’animation
- « Où est l’image ? » : enlever/renommer l’image et demander aux apprenants de corriger le chemin.
