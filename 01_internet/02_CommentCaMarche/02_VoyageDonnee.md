# 02 – Voyage de la donnée (client ↔ serveur)

Image simple : la lettre.
- Client = vous qui écrivez la lettre (votre recherche ou l’URL).
- Réseau = le facteur (câbles, fibre, wifi, 4G/5G) qui transporte la lettre.
- Serveur = le destinataire (ordinateur dans un data center) qui lit et renvoie la réponse.

Messages clés
- Internet, ce sont des ordinateurs qui se parlent.
- Un serveur est un ordinateur qui reste allumé 24/7 pour répondre.

Mini-démo
- Ouvrir un site et observer le chargement : une requête part, une réponse revient.
- Variante : débrancher le wifi, recharger la page pour montrer la dépendance au réseau.

Idée d’animation
- Faire dessiner le trajet : appareil → box → « nuage » → data center → retour. Chaque étape sur un post-it.
