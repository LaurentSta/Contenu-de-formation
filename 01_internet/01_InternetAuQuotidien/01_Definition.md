# 01 – Internet : définition

## Définition simple (niveau débutant)
Internet est un réseau mondial qui relie des millions d’ordinateurs entre eux afin d’échanger des informations.

## Définition un peu plus précise
Internet est une infrastructure mondiale de réseaux interconnectés permettant le transfert de données grâce à des protocoles comme HTTP ou HTTPS.

## Points clés à expliquer
- Internet relie des ordinateurs entre eux.
- Il permet d’envoyer et de recevoir des données.
- Il fonctionne grâce à des serveurs, des routeurs et des câbles sous-marins.
- Internet ≠ Web : le Web n’est qu’un service parmi d’autres (e-mail, FTP, streaming…).

## Quelques chiffres (utilisateurs dans le monde)
- 2010 : un peu plus de 2 milliards d’utilisateurs (estimation UIT).
- 2014 : presque 3 milliards.
- 2024 : env. 5,8 milliards (≈71 % de la population).
- 2025 : env. 6 milliards (≈74 %).
- 2026 (janv.) : autour de 6,0 milliards, la courbe progresse encore mais plus lentement.

## Exemple concret pour un slide
Quand vous consultez un site, votre appareil envoie une demande à un serveur situé ailleurs dans le monde, qui renvoie les informations.

## Question pédagogique possible
Quelle est la différence entre Internet et un site web ?

## Idées d’animation (présentiel / visio / 100 % distanciel)
- Nuage de mots « Internet c’est… » : post-it papier ou tableau collaboratif (Miro/Whiteboard/Pad).
- Jeu Vrai/Faux éclair : « Internet est stocké dans un nuage », « Internet = Web », « Les câbles passent sous la mer ».
- Schéma collectif : tracer le parcours d’une page web (appareil → box → réseau → serveur → réponse). À faire en binômes.
- Lexique minute : chacun reformule un mot (serveur, navigateur, lien, wifi) en langage simple pour le groupe.
